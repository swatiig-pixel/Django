# Django Todo App

A simple Todo List web application built using Django.

## Features
- Add tasks
- Mark tasks as completed
- Delete tasks
- Checkbox completion
- Background image UI

## Technologies Used
- Python
- Django
- HTML
- CSS

## Run the Project

pip install django  
python manage.py migrate  
python manage.py runserver  

Open: http://127.0.0.1:8000/
